# Around The World - Album Pessoal

Aplicação web interativa desenvolvida no bootcamp da TripleTen para praticar manipulação do DOM com JavaScript.

O projeto simula um álbum pessoal onde o usuário pode gerenciar cartões de lugares.

---

## Sobre o Projeto

Funcionalidades implementadas:
- Editar perfil
- Adicionar novos cartões
- Curtir cartões
- Remover cartões
- Visualizar imagem em modal

Principais conceitos aplicados:
- Manipulação do DOM
- `addEventListener` em botões e ações do usuario
- Manipulação de Popups com Classes (`classList`)
- Criação de blocos usando `<template>`
- Funções reutilizáveis para controle de modais

---

## Tecnologias Utilizadas
HTML5  
CSS3 (BEM)  
JavaScript (ES6+)
